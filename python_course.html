    Core Java & OOP Concepts

    Lay the foundation for back-end development with deep understanding of Java and OOP principles.

    • Java Basics: Syntax, data types, variables, operators, control statements
    • Object-Oriented Programming: Classes, objects, inheritance, polymorphism, encapsulation, abstraction
    • Advanced Java: Interfaces, abstract classes, inner classes, static/non-static blocks
    • Exception Handling, Multithreading, Lambda Expressions, Streams API
    • Java 8+ Features: Functional programming, method references, default/static methods in interfaces
    • Collections: List, Set, Map, Queue, Generics
    • Data Structures & Algorithms (DSA): Arrays, Linked Lists, Stacks, Queues, Trees, Sorting, Searching
    • Tools: Eclipse/IntelliJ IDEA, JDK 17+, Maven

    Spring Framework & RESTful APIs

    Build powerful back-end applications with Spring Boot and secure RESTful APIs.

    • Spring Core: IoC, Beans, Dependency Injection
    • Spring Boot: Auto-configuration, starters, annotations
    • Spring MVC: Controllers, REST endpoints, form validation
    • Spring Data JPA: Repository pattern, ORM with Hibernate
    • RESTful APIs: GET/POST/PUT/DELETE methods, error handling, request validation
    • Security: Spring Security, JWT tokens, OAuth2
    • Unit Testing: JUnit 5, Mockito, TestContainers
    • Tools: Postman, Swagger (OpenAPI), Lombok

    Database Design & MySQL

    Design robust relational schemas and integrate databases into backend systems.

    • Database Concepts: Tables, Primary & Foreign Keys, ACID properties
    • SQL: DDL, DML, aggregate functions, joins, views, triggers, stored procedures
    • Normalization: 1NF to 3NF, De-normalization
    • Advanced Queries: Subqueries, CTEs, indexing, performance tuning
    • NoSQL: Basics of MongoDB, document structure, CRUD operations
    • Integrating DBs with Java: Spring Data JPA, Hibernate ORM
    • Tools: MySQL Workbench, pgAdmin, MongoDB Compass

    React.js Frontend Development

    Build responsive, fast, and dynamic single-page applications (SPAs).

    • Frontend Basics: HTML5, CSS3, JavaScript (ES6+)
    • React Core: JSX, props, state, components, event handling
    • Advanced React: Hooks (useState, useEffect, useRef), Context API, Redux
    • Routing: React Router DOM
    • API Integration: Axios/Fetch, JSON handling, error boundaries
    • UI Libraries: Bootstrap, Material UI, Tailwind CSS (optional)
    • Component testing: Jest, React Testing Library
    • Tools: VSCode, Chrome DevTools, ESLint, Prettier

    Docker, Deployment & Career Prep

    Learn to deploy, manage, and scale applications, and get job-ready.

    • Containerization: Docker basics, Dockerfile, docker-compose
    • CI/CD Pipelines: GitHub Actions, Jenkins, testing & deployment automation
    • Deployment: Hosting apps on Heroku, AWS EC2, Netlify, Vercel
    • Monitoring & Logging: Prometheus, Grafana, ELK Stack (intro)
    • Project: End-to-end deployment of an E-Commerce app
    • Resume Building: Tailor-made for full stack roles
    • Interview Preparation: DSA questions, system design basics, HR tips
    • Mock Interviews & Career Guidance with industry mentors
